# Tomtit::Profile::Python

Tomtit scripts for Python development

# Install

    zef install Tomtit::Profile::Python

# Usage

    tom --list --profile Tomtit-Profile-Python
    tom  --profile Tomtit-Profile-Python

    # run new helpers
    tom tom python-pip-install-from-req
